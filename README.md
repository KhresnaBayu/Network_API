📚 Konsep yang Dipelajari
Proyek ini merupakan latihan praktik dari materi Pemrograman Mobile, mencakup:

✅ HTTP Request menggunakan Volley (JsonObjectRequest)

✅ Parsing data JSON (JSONObject, JSONArray)

✅ Komunikasi antar Activity dengan Intent & Extra

✅ RecyclerView GridLayoutManager yang responsif terhadap orientasi layar

✅ Penerapan SwipeRefreshLayout untuk refresh data

✅ Manajemen Loading State dengan ProgressBar

✅ Penanganan error koneksi dan parsing (Error Handling)

✅ Pemuatan gambar dari URL dengan Glide + placeholder

✅ Penggunaan View Binding untuk akses layout yang aman

✅ Izin jaringan (INTERNET, ACCESS_NETWORK_STATE) di AndroidManifest
